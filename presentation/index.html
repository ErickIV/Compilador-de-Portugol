Compilador de Portugol

Um Tradutor Educacional de Portugol para Python

Acadêmico: Erick Vieira, Patrick Lohn, Rafael Sonoki

Curso: Ciência da Computação

O Problema

A barreira de entrada na programação.

  • Sintaxe complexa de linguagens industriais (C, Java).
  • Barreira do idioma (Inglês).
  • Dificuldade em focar na lógica pura.

A Solução

Unir a simplicidade do Portugol com o poder do Python.

Portugol escreva("Olá")
Python print("Olá")

Frontend: Análise

Analisador Léxico

Transforma texto em tokens usando Expressões Regulares e Autômatos Finitos.

Analisador Sintático

Constrói a Árvore Sintática Abstrata (AST) usando descida recursiva.

Analisador Semântico

Valida tipos, escopo de variáveis e inicialização.

Backend: Síntese e Otimização

  • Otimizações

    Constant Folding, Dead Code Elimination, Copy Propagation.

  • Geração de Código

    Tradução final para Python 3.11+.

Otimizações Implementadas

5 tipos de otimizações automáticas

Constant Folding

Avalia expressões constantes em tempo de compilação

5 + 3 * 2  →  11

Algebraic Simplification

Aplica identidades matemáticas

x + 0  →  x
x * 1  →  x
x * 0  →  0

Constant Propagation

Substitui variáveis por valores conhecidos

x = 5
y = x + 3  →  y = 8

Dead Code Elimination

Remove código nunca utilizado

temp = x + y  // Removido
